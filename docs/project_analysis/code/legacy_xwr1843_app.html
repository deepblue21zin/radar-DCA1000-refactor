놓치기 쉬운 부분

코드를 읽을 때 자주 헷갈리거나, 설명 없이 넘어가면 오해하기 쉬운 부분을 따로 정리했다.

  • 이 파일은 현재 추천 entrypoint가 아니다. 참고용과 비교용으로 읽는 편이 맞다.
  • 한 파일에 제어와 렌더와 워커 관리가 같이 있어, 구조적 단순함과 유지보수 어려움이 동시에 보인다.
  • 현재 문서와 리포트 체계는 이 파일이 아니라 live_motion_viewer 경로 기준으로 발전했다.
  • 새 기능을 붙일 땐 이 파일보다 현재 앱 구조를 기준으로 생각하는 편이 안전하다.

26def send_cmd(code):
27 # command code list
28 CODE_1 = (0x01).to_bytes(2, byteorder='little', signed=False)
29 CODE_2 = (0x02).to_bytes(2, byteorder='little', signed=False)
30 CODE_3 = (0x03).to_bytes(2, byteorder='little', signed=False)
31 CODE_4 = (0x04).to_bytes(2, byteorder='little', signed=False)
32 CODE_5 = (0x05).to_bytes(2, byteorder='little', signed=False)
33 CODE_6 = (0x06).to_bytes(2, byteorder='little', signed=False)
34 CODE_7 = (0x07).to_bytes(2, byteorder='little', signed=False)
35 CODE_8 = (0x08).to_bytes(2, byteorder='little', signed=False)
36 CODE_9 = (0x09).to_bytes(2, byteorder='little', signed=False)
37 CODE_A = (0x0A).to_bytes(2, byteorder='little', signed=False)
38 CODE_B = (0x0B).to_bytes(2, byteorder='little', signed=False)
39 CODE_C = (0x0C).to_bytes(2, byteorder='little', signed=False)
40 CODE_D = (0x0D).to_bytes(2, byteorder='little', signed=False)
41 CODE_E = (0x0E).to_bytes(2, byteorder='little', signed=False)
42
43 # packet header & footer
44 header = (0xA55A).to_bytes(2, byteorder='little', signed=False)
45 footer = (0xEEAA).to_bytes(2, byteorder='little', signed=False)
46
47 # data size
48 dataSize_0 = (0x00).to_bytes(2, byteorder='little', signed=False)
49 dataSize_6 = (0x06).to_bytes(2, byteorder='little', signed=False)
50
51 # data
52 data_FPGA_config = (0x01020102031e).to_bytes(6, byteorder='big', signed=False)
53 data_packet_config = (0xc005350c0000).to_bytes(6, byteorder='big', signed=False)
54
55 # connect to DCA1000
56 connect_to_FPGA = header + CODE_9 + dataSize_0 + footer
57 read_FPGA_version = header + CODE_E + dataSize_0 + footer
58 config_FPGA = header + CODE_3 + dataSize_6 + data_FPGA_config + footer
59 config_packet = header + CODE_B + dataSize_6 + data_packet_config + footer
60 start_record = header + CODE_5 + dataSize_0 + footer
61 stop_record = header + CODE_6 + dataSize_0 + footer
62
63 if code == '9':
64 re = connect_to_FPGA
65 elif code == 'E':
66 re = read_FPGA_version
67 elif code == '3':
68 re = config_FPGA
69 elif code == 'B':
70 re = config_packet
71 elif code == '5':
72 re = start_record
73 elif code == '6':
74 re = stop_record
75 else:
76 re = 'NULL'
77 # print('send command:', re.hex())
78 return re
